# White Christmas in Germany 

<img src="white_christmas_viz.png" alt="White Christmas Visualization" width="600">

Data was extracted from the [DWD](https://opendata.dwd.de/climate_environment/CDC/observations_germany/climate/daily/kl/historical/) and stored in a SQLite Database. 
The visualization was created using Python (geopandas, matplotlib) and post-processed in Inkscape.
